The To Do List

The To Do list is accessed from the Action/To Do menu choice from the main Month screen. This list shows all appointments marked as To Do's that have not been marked as done. To Do's can be marked as done from the To Do window menu. There are 2 options to pick from when marking To Do's as done. The Done (No Delete) menu option, removes the appointment from the To Do list, but leaves it on the calendar. The Done (Delete) option removes it from the calendar also (unless the todo has more repeats, in which case the next occurrence is shown on the to do list).



When a To Do is marked as done, it is shown with a strike-through font, like this: done

There is an option that hides all strike-through items on the calendar to reduce clutter.
